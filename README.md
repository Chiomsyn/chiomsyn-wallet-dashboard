Chiomsyn Wallet Dashboard

A minimal crypto wallet interface built with Next.js and ethers.js.  
This project explores core blockchain interactions including wallet connection, balance retrieval, transaction history, and sending transactions.

Features:

- MetaMask wallet connection
- Ethereum (Sepolia) balance display
- Transaction history via Etherscan API
- Send ETH transactions
- Network detection

Tech Stack:

- Next.js
- TypeScript
- ethers.js

Motivation:
Built to understand how wallet providers interact with frontend applications and to explore building intuitive crypto user experiences.
